Ecommerce Sales Analysis Project

## 🔍 Project Overview
This project focuses on analyzing ecommerce sales data to extract meaningful insights and support business decision-making. The analysis is performed using Excel, SQL, Python, and Power BI.

## 🛠️ Tools & Technologies Used
- Excel (Data Cleaning & Pivot Tables)
- SQL (Data Analysis & Queries)
- Python (Pandas, Matplotlib for Data Analysis & Visualization)
- Power BI (Interactive Dashboard)

## 📁 Project Structure
- **Dataset/** → Raw dataset used for analysis
- **SQL_Queries/** → SQL queries for analysis
- **Outputs/** → Results generated from SQL queries
- **Python/** → Jupyter Notebook with data analysis and visualization
- **Dashboard/** → Power BI dashboard file

## 📈 Key Insights
- Identified top-performing products and categories
- Analyzed sales and profit across regions
- Detected loss-making products
- Found top customers contributing to revenue
- Monthly sales trend analysis

## 📊 Dashboard
Interactive Power BI dashboard provides:
- Sales by Region
- Profit Analysis
- Category-wise Performance
- Customer Insights

## 🎯 Conclusion
This project demonstrates end-to-end data analysis workflow from raw data to insights and visualization.
